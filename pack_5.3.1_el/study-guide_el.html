Οδηγός μελέτης

Ποιος είναι ο καλύτερος τρόπος να χρησιμοποιήσεις το LenMus Phonascus?

Ο καλύτερος τρόπος χρήσης του LenMus Phonascus είναι να ως συμπλήρωμα/βοήθημα στις μουσικές σπουδές σου, για μελέτη στο σπίτι των εννοιών που εξήγησε ο δάσκαλος στο σχολείο/ωδείο. Συνήθως στο σχολείο/ωδείο υπάρχει πολύ λίγος χρόνος για ασκήσεις και ο σπουδαστής πρέπει να εξασκείται στο σπίτι. Όμως μερικές ασκήσεις, ιδιαίτερα όσες σχετίζονται με ακουστική εξάσκηση, είναι δύσκολο να γίνονται σπίτι χωρίς δάσκαλο. Είναι σε αυτές τις περιπτώσεις, που το LenMus μπορεί να βοηθήσει σε μεγάλο βαθμό.

Είναι σημαντικό να γίνει ξεκάθαρο ότι το LenMus Phonascus δεν μπορεί να αντικαταστήσει ένα καλό δάσκαλο μουσικής. Παρ' όλο που μπορεί υποστηρίξει και να συμπληρώσει τη διδασκαλία της μουσικής, το LenMus δεν μπορεί να αντικαταστήσει την κατανόηση και την προσωπική επαφή του ζωντανού δασκάλου που σε καθοδηγεί. Το LenMus Phonascus είναι σπουδαίο εργαλείο για μελέτη στο σπίτι, ειδικά στην ακουστική εξάσκηση, αλλά ένας καλός δάσκαλος είναι αναντικατάστατος και πραγματικά θα επιταχύνει τη μάθησή σου.

Αν δεν έχεις τη δυνατότητα να έχεις δάσκαλο, το LenMus Phonascus θα μπορούσε να σε βοηθήσει να μάθεις θεωρία της μουσικής και να αναπτύξεις ικανότητες μουσικής αντίληψης, αλλά χωρίς δάσκαλο θα πρέπει να είσαι επιπρόσθετα κριτικός προς τον εαυτό σου. Είναι εύκολο να νομίζεις ότι ξέρεις κάτι χωρίς να το ξέρεις πραγματικά.

Χρησιμοποιώντας το βιβλίο Γενικών Ασκήσεων

Το βιβλίο Γενικών Ασκήσεων προορίζεται να σε βοηθήσει να εξασκήθείς σπίτι σε αυτά που έμαθες στο σχολείο. Είναι οργανομένο με τέτοιο τρόπο, ώστε να μπορείς να βρεις γρήγορα την κατάλληλη άσκηση για αυτό που θέλεις να εξασκήσεις. Αυτό το βιβλίο δεν είναι κατάλληλο για μελέτη άνευ-διδασκάλου, χωρίς να ακολουθείς κάποιο πρόγραμμα σπουδών και κάποιο βιβλίο θεωρίας της μουσικής.

Αυτό το βιβλίο περιέχει όλες τις ασκήσεις οι οποίες είναι διαθέσιμες στο LenMus Phonascus. Οι ασκήσεις είναι ταξινομιμένες κατά είδος/τύπο χωρίς να ακολουθούν κάποια λογική σειρά για μελέτη, και δεν υπάρχουν επεξηγήσεις για τον σκοπό της κάθε άσκησης. Κάθε άσκηση μπορεί να ρυθμιστεί, ώστε να μπορείς να εξασκήσεις εκείνα τα σημεία στα οποία έχεις περισσότερες δυσκολίες.

Όταν έχεις δάσκαλο ο οποίος σου εξηγά τη θεωρία, το βιβλίο Γενικών Ασκήσεων είναι, ίσως, το καλύτερο βοήθημα για τις σπουδές σου. Αν όμως δεν κάνεις μαθήματα μουσικής, αυτό το βιβλίο δε θα σου προσφέρει πολλά, αφού περιέχει μόνο ασκήσεις χωρίς επεξηγήσεις. Σε αυτή την περίπτωση καλύτερα να χρησιμοποιήσεις κάποιο από τα άλλα βιβλία που παρέχονται με το LenMus Phonascus.

Χρησιμοποιώντας τα μαθήματα Μουσικής Ανάγνωσης

Απαιτούνται πολλά χρόνια εξάσκησης για να μάθει κανείς να διαβάζει μουσική ("εκ πρώτης όψεως ανάγνωση" ή "πρίμα βίστα"). Στην πραγματικότητα δεν είναι τόσο δύσκολο να διαβάζεις μουσική, αλλά παίρνει χρόνο, υπομονή και εργασία. Δεν υπάρχουν μαγικές προσεγγίσεις. Είναι όπως όταν μαθαίνουμε ανάγνωση. Θυμάσαι; Σου χρειάστηκαν πολλά χρόνια εξάσκησης!

Τα βιβλία Μουσικής Ανάγνωσης είναι διαβαθμισμένα και περιέχουν εισςαγωγικά κείμενα ώστε να μπορείς να μάθεις να διαβάζεις μουσική χωρίς δάσκαλο. Πάραυτα, συστήνεται να πάρεις τουλάχιστον κάποια αρχική καθοδήγηση από κάποιο δάσκαλο σχετικά με το σωστό τρόπο εξάσκησης στην εκ πρώτης όψεως ανάγνωση.

Αν μελετάς χωρίς δάσκαλο, πρέπει να ακολουθείς τα μαθήματα με τη σειρά. Μόλις διαβάσεις τα εισαγωγικά μαθήματα του αρχικού επιπέδου 1, αρχίζουν οι ασκήσεις. Πάρε τα μαθήτα ένα-ένα με τη σειρά, και δούλεψε τις ασκήσεις μέχρι να εμπεδόσεις καλά κάθε μάθημα. Προχώρα με το δικό σου ρυθμό. Κάποια μαθήματα μπορεί να χρειαστούν μόνο μια-δυο μέρες να εμπεδοθούν. Πολλά θα χρειαστούν μια εβδομάδα ή περισσότερο. Απλά έχονταες διαβάζει ένα μάθημα δεν σημαίνει ότι το εμπέδοσες. Πρέπει να κάνεις εξάσκηση ... πολλή εξάσκηση! Οι ασκήσεις σε κάθε μάθημα σε αναγκάζουν να χρησιμοποιήσεις και τους ρυθμούς που μελετησες σε προηγούμε να μαθήματα. Γιαυτό, όταν επικεντρώνεσαι στο παρών μάθημα θα αναθεωρείς ταυτόχρονα και όλα τα προηγούμενα. Κάθε βιβλίο Μουσικής Ανάγνωσης αναμένεται να διαρκέσει μια ακαδημαϊκή χρονιά.

Αν παρακολουθείς τακτικά μαθήματα, τα βιβλία Μουσικής Ανάγνωσης θα σου είναι ιδιαίτερα χρήσιμα για δύο σκοπούς:

  1. Για να μάθεις, ακουστικά, τον ήχο και ρυθμό κάθε ομάδας φθόγγων.
  2. Για να έχεις περισσότερες ασκήσεις για εξάσκηση. Συνήθως, όταν μελετάς εκ πρώτης όψεως ανάγνωση, μελετάς τις ίδιες ασκήσεις πολλές φορές και οι ασκήσεις απομνημονεύονται. Έτσι γίνεται δύσκολο να εξσκηθείς στην ανάγνωση. Τα βιβλία Μουσικής Ανάγνωσης είναι μια αστείρευτη πηγή ασκήσεων για μελέτη. Η παρτιτούρα της κάθε άσκησης δημιουργείται αυτόματα κάθε φόρα που πατάς το κουμπί 'Νέα άσκηση'. Γιαυτό, οι ασκήσεις ποτέ δεν επαναλαμβάνονται.

Χρησιμοποοιώντας τα μαθήματα Θεωρίας και Αρμονίας

Στο βιβλίο Θεωρίας και Αρμονίας θα βρεις τα μαθήματα θεωρίας οργανωμένα με λογική κατά θέμα. Προσπάθησα μαζέψω υλικό θεωρίας της μουσικής, κατά θέμα (π.χ. κλίμακες ή πτώσεις). Ακόμα προσπάθησα να συμπεριλάβω ασκήσεις και παραδείγματα για εξάσκηση στις βασικές έννοιες του μαθήματος.

Αυτό το βιβλίο δεν είναι κατάλληλο για μελέτη χωρίς να ακολουθείς ένα πρόγραμμα σπουδών. Όταν μαθαίνεις μουσική, δε μαθαίνεις πρώτα τα πάντα για ένα θέμα και μετά να προχωρήσεις στο επόμενο θέμα. Αντίθετα, μαθαίνεις αργά λίγο από την κάθε έννοια σε κάθε επίπεδο. Στο επόμενο επίπεδο αναθεωρείς προηγούμενα έννοιες και εμβαθύνεις στο θέμα. Όμως το βιβλίο Θεωρίας και Αρμονίας δεν είναι χωρισμένο σε επίπεδα. Κάθε έννοια είναι οργανωμένη σε μαθήματα και θα πρέπει να σταματήσεις να προχωράς σε μια έννοια όταν εκπληρώσεις τις απαιτήσεις του προγράμματος σπουδών το οποίο ακολουθείς.
